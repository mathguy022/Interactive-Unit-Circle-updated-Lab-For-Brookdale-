Interactive Unit Circle & Trigonometric Identity Lab Pro
Made by Ahmed Said • Made for Brookdale Community College

A classroom-ready, interactive web application designed for Precalculus and Trigonometry students.

Overview

This tool provides a dynamic visual representation of the unit circle, allowing students to explore the relationships between angles, coordinates, and the six trigonometric functions in real time. It bridges geometric intuition with algebraic computation, making abstract concepts tangible and easy to understand.

Key Features


Angle Exploration


Interactive angle slider (0°–360°) with live updates
Degrees / Radians toggle — all labels update instantly
Snap to Common Angles mode for the 17 standard angles (0°, 30°, 45°, 60°, 90°, ... 360°)
Draggable point on the unit circle — click and drag (mouse or touch) to any angle
Quick Angle Select grid with Degrees/Radians toggle for jumping to standard angles
Keyboard navigation — arrow keys (Shift = 15° steps), Spacebar = Play/Pause
Animation
Play / Pause / Resume / Reset controls
Smooth requestAnimationFrame-based rotation at 60 FPS
Adjustable speed (1–10° per frame)
Visualization Elements (toggleable)
Show / Hide Unit Circle Body (axes and angle handle remain visible)
Show / Hide Common Angles (all 17 standard points with labels)
Show / Hide Coordinates on common angle points
Show / Hide Reference Triangle (color-coded legs)
Show / Hide Projections (cosine on x-axis, sine on y-axis)
Show / Hide Tangent Line at the current point
Show / Hide Angle Arc with central angle label


Trigonometric Values


Displays sin, cos, tan, csc, sec, cot in real time
Exact Values mode — shows symbolic forms (√2/2, √3/2, 1/2, etc.) for standard angles
Decimal Values mode — shows 4-decimal approximations for any angle
Live coordinate display (x, y) in exact or decimal form
Quadrant indicator and reference angle display
Degree ↔ Radian conversion always visible
Identity Visualization (Right Panel)
Animated Pythagorean Identity bar: sin²θ + cos²θ = 1
Live function value bars for sin, cos, tan
Related identities reference (1 + tan²θ = sec²θ, double-angle formulas, etc.)
Signs by Quadrant table — color-coded (green = positive, red = negative) showing sin, cos, tan, cot for all four quadrants
Design & Accessibility
Clean, modern educational interface inspired by Desmos, GeoGebra, and Khan Academy
Highly visible black axes and labels for classroom projection
Fully responsive — works on desktop, tablet, and mobile
Keyboard navigation and screen reader support (ARIA labels)
Print-friendly layout
